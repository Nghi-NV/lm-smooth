## 0.1.0

- Initial public release of `lm_smooth`.
- Added `SmoothGrid` for high-performance masonry/staggered layouts with fixed column count.
- Added precomputed layout pipeline backed by `LayoutCache` and `SpatialIndex`.
- Added optional isolate-based layout computation for very large datasets.
- Added long-press drag reorder support with preview translation and edge auto-scroll.
- Added example app demonstrating large datasets, column switching, and reorder mode.
- Added unit and widget tests for layout, gesture handling, reorder preview, and auto-scroll.
